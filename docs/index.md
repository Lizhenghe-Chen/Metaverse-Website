# 香港科技大学（广州）元宇宙联合创新实验室

本网站展示了实验室在元宇宙、数字人、虚拟现实和人工智能等领域的研究与应用成果，涵盖以下方向：

1. 基于大规模模型智能体的数字人技术
2. 虚拟空间中的多感官人机交互技术
3. 基于人体数据实时采集的虚拟体现技术
4. 基于人工智能的元宇宙内容生成技术
5. 元宇宙教育的社会适应性研究

## 功能和产品

### Unity轻量级无人机仿真系统

#### VR无人机虚拟仿真和培训

基于Unity引擎开发多无人机遥操作系统，支持单/多无人机飞行控制、绳索货物运输、用户交互界面、实时数据可视化和任务模拟。该系统为无人机协同运输研究和应用提供高效直观的实验平台，可用于训练无人机操作能力和飞控知识。

#### 虚拟现实无人机环游世界和航拍

结合VR引擎与地图服务，开发虚拟现实无人机遥控系统，实现沉浸式无人机遨游体验，提升用户空间感知和操作能力。

#### 虚拟现实无人机灭火

在无人机仿真基础上，增加灭火功能，探索无人机在应急救援等场景的应用潜力。

### VR多人农场经济学博弈实验

支持多人参与的虚拟农场经济学实验，研究群体行为与经济决策。

### VR虚拟沙盘讲解

通过虚拟现实技术实现沙盘讲解和互动，提升教学与展示效果。

### VR虚拟现实拼抢和射击训练

开发虚拟现实环境下的拼抢与射击训练系统，提升用户反应能力和团队协作。

### AI全息及多终端数字人框架

构建支持多终端的AI数字人系统，实现全息交互与智能服务。

### 多人网络框架

支持多人协作与互动的网络架构，适用于各类虚拟现实应用。

#### VR多人军事无人机活动

开发面向军事训练的VR无人机协作平台，支持多用户参与。
