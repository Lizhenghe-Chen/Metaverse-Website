# 3. UI面板模块（AddonPanels）
**模块标识符**：AddonPanels.py  
**编制员**：开发团队  
**修改完成日期**：2025年12月31日

### 3.1 模块功能
定义系统用户界面面板，提供直观操作界面，包含：服务器连接、模型选择、角色模型、目标骨架、动作描述、参数设置、操作按钮、状态显示。

### 3.2 面板布局
采用分组框布局，共8个区域：
1. 服务器连接：IP/端口输入、连接按钮、状态显示
2. 模型选择：下拉菜单、提示、选中模型名
3. 角色模型：目录选择、文件列表、预览按钮
4. 目标骨架：骨架对象选择框
5. 动作描述：多行文本、单词计数
6. 参数设置：帧数、IK、自动烘焙、输出目录
7. 操作按钮：生成、重导入、重定向烘焙、全流程
8. 服务器状态：进度查询、状态/进度显示

### 3.3 操作方法
1. 3D视图按N键打开侧边栏，切换到"motionai"
2. 输入IP端口，连接服务器至在线
3. 选择动作生成模型
4. 指定模型目录，预览加载角色
5. 选择目标骨架
6. 输入英文动作描述
7. 设置参数
8. 点击操作按钮执行
9. 查询任务进度
