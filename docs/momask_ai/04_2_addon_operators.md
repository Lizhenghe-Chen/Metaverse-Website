# 2. 操作符模块（AddonOperators）
**模块标识符**：AddonOperators.py  
**编制员**：开发团队  
**修改完成日期**：2025年12月31日

### 2.1 模块功能
定义MoMask系统所有Blender操作符，实现UI与业务逻辑桥接。包含：模型预览、热加载、BVH导入、动作重定向、完整管道操作。

**对应需求**：软件需求说明书第3章第3节"用户交互功能"

### 2.2 输入输出信息
- **输入**：外部模型目录、模型枚举、目标骨架、BVH目录、操作符参数、烘焙后清除标志
- **输出**：加载的集合/骨架、导入的骨架、操作结果、用户提示

### 2.3 处理描述
- **模型预览**：验证选择→加载模型→移至中心→设为目标骨架
- **模型热加载**：构建路径→验证文件→加载骨架→自动指定目标
- **BVH重导入**：确定目录→扫描文件→导入BVH
- **动作重定向**：验证参数→导入BVH→执行重定向→烘焙动画→清理约束
- **完整管道**：请求下载→获取路径→重定向预览→自动烘焙
