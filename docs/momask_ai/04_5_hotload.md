# 5. 热加载工具模块（hotload）
**模块标识符**：hotload.py  
**编制员**：开发团队  
**修改完成日期**：2025年12月31日

### 5.1 模块功能
实现.blend文件热加载，支持加载骨架/集合，包含：文件扫描、骨架加载、集合加载、智能识别。

### 5.2 输入输出信息
- **输入**：文件路径、链接模式（True=链接/False=导入）
- **输出**：加载的骨架对象或集合

### 5.3 处理描述
1. 打开文件（Blender库链接）
2. 遍历查找ARMATURE对象与含骨架集合
3. 优先加载骨架，无则加载集合
4. 返回加载结果

### 5.4 有关事项
- 支持相对/绝对路径
- link=True：创建链接，原文件更新自动同步
- link=False：创建副本，独立使用
